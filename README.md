# Gestión de colas por tickets (Angular 20 + CodeIgniter 4.6 + MySQL)

## Estructura
- `backend/`: API REST (CodeIgniter 4.6) con JWT y migraciones
- `frontend/`: Web (Angular 20 standalone) con Guards y servicios
- `docs/sql/schema.sql`: Script SQL (MySQL)

## Arranque rápido
1) Base de datos: ejecuta `docs/sql/schema.sql` o usa migraciones del backend.
2) Backend: ver `backend/README.md`.
3) Frontend: ver `frontend/README.md`.
